Assignment 2 - features
1. Shows graphs for weight, height, and GPA
2. The colour for font and border matches to the colour the student chose
3. Boxes of students are lined up in one line and aligned at the bottom
4. If boxes of students cannot fit in a page, it can be scrolled
5. When go to 'show', ranking is shown there
6. When create a new student or edit a student, it gives error messages and re-direct to the form
    if some fields are blank
7. For weight, height, and GPA, decimals are allowed, input type number is used for their forms
    so you can use up/down arrow to change the values
8. Added a column for the gender of student
9. The background colours of edit and show pages change depending on the gender of students
10. Added the link for sample colours
